# Contributing to GreenArchitect-TR

First off, thank you for considering contributing to the GreenArchitect-TR TEKNOFEST project! It's people like you that help make the world a greener place through technology.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker.
- Describe the bug and steps to reproduce.
- Include environment details (OS, software versions).

### Suggesting Enhancements
- Open an Issue with the tag `enhancement`.
- Explain why this feature would be useful for TEKNOFEST participants.

### Pull Requests
1. Fork the repo.
2. Create a new branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Coding Standards
- Maintain a professional tone.
- Ensure all technical descriptions are accurate and follow industry standards (ISO, MQTT, etc.).
- Use clear, descriptive names for files and variables.
